## Setup
Install [Git](https://git-scm.com/).

    $ git clone https://github.com/gdegas/Surfazon.git
    
Open 'index.html' in your prefered browser.

## Technologies Used
Surfazon was made with:
+ JavaScript
+ CSS3
+ HTML5
+ Bootstrap 3
+ DOM
+ Event Listeners
+ Responsive Design
